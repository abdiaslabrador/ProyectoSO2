Monitor escrito en Powershell como proyecto para la cátedra de Sistemas Operativos II.

	Esta versión titulada "con_hijos" a diferencia de la versión "master" suma la memoria ram de todas todos los procesos con el mismo nombre, es decir, si tenemos 5 bloc de notas abiertos el "Visor de procesos" sumará la memoria ram de cada uno y los unirá en una sola fila.

	Al darle en terminar proceso se eliminaran todos lo procesos con ese nombre. Cabe destacar que la id que se muestra es de un proceso real, más no de un proceso en conjunto de todos, me refiero a que existe un proceso con ese id pero la memoria mostrada no es la memoria de ese proceso, al menos que solo exista un solo proceso de ese tipo abierto.
